# GUI quick start

This guide explains how to run the Nullspace GUI client locally.

## Prerequisites

- A working Rust toolchain (stable), including Cargo.

## Run the app

1. Clone the repository:

   ```sh
   git clone https://github.com/mel-project/nullspace.git
   cd nullspace
   ```

2. Run the GUI client:

   ```sh
   cargo run -p nullspace-egui
   ```
